# Week 2 Activity 1 – OOP Calculator

## Description
This project is basic mathematical operations, using OOP in Python.

## OOP Conceps
- **Class**: A `Calculator` class is created for all operations.
- **Methods**: The class contains methods such as `add()`, `subtract()`, `multiply()`, and `divide()`.
- **Functions**: Separate functions (`get_numbers()` and `choose_operation()`) handle user input and interaction.

## Features
- Addition
- Subtraction
- Multiplication
- Division (with error handling)
